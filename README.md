# GraphqlAngularDemo
```
yarn
docker-compose up
yarn start:backend
```


#### Getting Started
1. Clone the repo  `git clone https://github.com/ridhamtarpara/go-graphql-demo.git`
2. Run `dep ensure`
3. Run `go run server/server.go`
4. Open `http://localhost:8080/` for GraphQL Playground
